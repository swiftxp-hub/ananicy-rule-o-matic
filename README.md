## Installation

### Prerequisites
* Rust toolchain (cargo)

### Build from Source

```bash
git clone https://github.com/swiftxp-hub/ananicy-rule-o-matic.git
cd ananicy-rule-o-matic
cargo build --release
```

The binary will be located at `target/release/ananicy-rule-o-matic`.

## Usage

Run the tool from your terminal.

```bash
./ananicy-rule-o-matic [OPTIONS]
```
